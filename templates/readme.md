# Templates

Plantillas de referencia para el workflow de Flutter.

## Estructura

```
templates/
├── templates/          # Plantillas base
```

## Plantillas (templates/)

| Archivo | Descripción |
|---------|-------------|
| `agent_templates.md` | Reglas del agente y protocolo de calidad |
| `feature_templates.md` | Templates para nuevas features (SPEC, ARCHITECTURE, TODO) |
| `project_templates.md` | Template unificado para PROJECT_LOG.md |

## Uso

Copiar las plantillas según necesidad del proyecto. Ver `createproject.md` para el flujo completo de configuración.

> **Nota:** Los ejemplos de código están en la carpeta `examples/` en la raíz.
> Los skills están en la carpeta `skills/` en la raíz.
